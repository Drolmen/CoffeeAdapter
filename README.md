## 设计理念
**CoffeeAdapter**被设计为在一个单独的.java文件中使用。不建议以匿名内部类的方式，在Activiy、Fragment中使用。

## 添加依赖
**Gradle:**
```
compile 'com.drolmen:CoffeeAdapter:0.1.0'
```
**Maven**
```
<dependency>
  <groupId>com.drolmen</groupId>
  <artifactId>CoffeeAdapter</artifactId>
  <version>0.1.0</version>
  <type>pom</type>
</dependency>
```
